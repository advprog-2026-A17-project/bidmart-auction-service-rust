# Spesifikasi — Modul 3 (Lelang dan Penawaran)

## Tanggung jawab

Aturan penawaran, anti-sniping, penutupan lelang, integrasi hold dompet, dan publikasi event outbox.

## Fitur

- Domain model: Money, status sesi, validasi increment & self-bid
- Anti-sniping (perpanjangan 2 menit di window akhir)
- Place bid dengan hold dompet (sinkron)
- Penutupan WON/UNSOLD + release/convert hold
- Transactional outbox → RabbitMQ (`auction.*.v1`)
- Scheduler pending closure & outbox publisher
- Tipe lelang `ENGLISH` (reserve); ekstensi SCHOLARSHIP/ENTERPRISE disiapkan via `CloseStrategy`
- Profiling before/after (domain optimization)

## Integrasi

| Modul | Arah | Mekanisme |
|-------|------|-----------|
| 2 Katalog | Keluar/Masuk | HTTP validasi + event bid/ended |
| 4 Dompet | Keluar | HTTP hold/release/convert |
| 5 Order | Keluar | Event outbox |
